# 城市认知智能挑战赛代码说明

```
code/
├── loopga.sh
├── data/
│   ├── map.json	// 地图文件，此代码中已删去
│   ├── od.txt	// 车流文件，此代码中已删去
├── process/
│   ├── draw/		// 绘制地图工具
│   │   ├── out720/		// 仿真结果文件夹
│   │   │   ├── time.txt	// 仿真时间文件，此代码中已删去
│   │   │   ├── traj.txt	// 仿真轨迹文件，此代码中已删去
│   │   ├── 720.txt	// 限行规则文件
│   │   ├── get_car_num.py	// 从轨迹文件中获取每条车道上的车辆数
│   │   ├── get_data.sh	// 仿真命令脚本
│   ├── ga/		// 遗传算法
│   │   ├── curr_data.py	// 从数据库中取出排名前40的样本
│   │   ├── single.py			// 对于一对双亲的遗传算法
│   │   ├── ga.py		// 遗传算法主入口
│   │   ├── ga_init.sh		// 遗传算法初始化脚本
│   │   ├── run_gax.sh(x=1~16)	// 同时开16个仿真
│   │   ├── run.sh		// 运行遗传算法迭代
│   ├── randomsel/		// 用随机数生成的方式得到初始种群
│   ├── utils/		// 对于map.json的探索，比如筛选左转车道、快速车道等
├── GAdata/
│   ├── getallrank.py		// 分析数据库的样本及天梯
│   ├── getfirst.py		// 对于每一轮的样本进行排序，显示天梯
│   ├── gadata/		// 数据库，原始数据已去除，解释见算法说明文档
│   ├── garaw/		// 数据库，原始数据已去除，解释见算法说明文档
│   ├── raw_data/		// 数据库，原始数据已去除，解释见算法说明文档
```
